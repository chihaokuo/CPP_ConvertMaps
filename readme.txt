This tool is to convert map files from http://www.movingai.com/benchmarks/

How to use:

Put all maps in the folder/subfolder. Then run Allmaps.bat to create RunConvert.bat.
Then run RunConvert.bat to get all txt files.

Default max map size is 128. Map bigger than 128x128 won't be converted.
Change the setting in Allmaps.bat.
